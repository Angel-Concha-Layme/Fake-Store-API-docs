## Features

<CardGrid stagger>
	<Card title="Todas las Operaciones CRUD" icon="pencil">
		Aprovecha nuestra API para realizar sin limitaciones todas las operaciones CRUD (Crear, Leer, Actualizar y Eliminar).
    </Card>
	<Card title="Paginación" icon="document">
		Soporta paginación a través de los métodos 'size' y 'limit', permitiendo características como el desplazamiento infinito.
	</Card>
	<Card title="Auth con JWT" icon="setting">
		Utiliza JWT para tokens de acceso y actualización, y prueba de manera segura los endpoints con este método de autenticación.
	</Card>
	<Card title="Filtros" icon="magnifier">
		Filtra productos por precio, título y categoría, lo que permite la creación de interfaces de usuario sofisticadas basadas en estos criterios.
	</Card>
  <Card title="Rest API" icon="star">
		La API emplea la arquitectura API Rest, la estructura más ampliamente adoptada en la industria.
	</Card>
  <Card title="Postam e Insomia" icon="random">
		Se proporciona archivos para ambos, Postman e Insomnia, permitiéndote probar fácilmente cada endpoint.
	</Card>
    <Card title="Imágenes con Inteligencia Artificial" icon="sun">
